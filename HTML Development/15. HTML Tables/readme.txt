<table> ... </table> is the syntax for table tag usage.

The attributes used are explained as follows:

Width: defines the width of the table by default as per pixels and can also be expressed in percentage.

Border: specifies width of the border around the table.

Bordercolor: specifies border color of the respective table.

Cellspacing: specifies the space between the cells.

Cellpadding: specifies space between the cell wall and the cell content.
